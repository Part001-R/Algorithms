Пример работы с сессией.

Пользователь проходит регистрацию. Создаётся хэш токена сессии из имени пользователя и пароля.
{
    "user_name": "Alex",
    "user_pwd": "AAA",
    "user_pwd_repeat": "AAA"
}

Пользователь проходит аутентификацию. Пользователю, в качестве токена сессии, возвращается хэш.
{
    "user_name": "Alex",
    "user_pwd": "AAA"
}

Пользователь запрашивает данные.
